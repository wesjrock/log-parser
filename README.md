# Required
1. Install Ruby (version 3.1.0): https://rvm.io
2. Install Rspec and Json support: Run the command 'bundle'

# Running the Aplication
Go to root directory
```
log-parser
```
Run
```
ruby main.rb
```

# Running Tests
Go to directory
```
log-parser/spec/lib
```
Run
```
rspec parse_spec.rb
```
